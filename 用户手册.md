## 系统概述

Sales Proxy 是一个基于 Cloudflare Workers 的药品代销管理平台，专为医药行业合作伙伴设计。该系统提供了一套完整的解决方案，用于管理合作条款、跟踪运营流程、共享数据以及优化营销策略。

### 系统特点
- 响应式设计，支持多种设备访问
- 实时数据可视化仪表盘
- 多维度运营指标监控
- 智能选品和内容生成
- 合规性检查和风险控制
- 完整的合作流程管理

## 操作指南

### 登录系统
1. 打开浏览器，访问系统URL：https://salesproxy.518166.com.cn/
2. 输入用户名和密码
3. 点击"登录"按钮
4. 登录成功后将自动跳转到仪表盘页面

### 仪表盘使用

#### 筛选器操作
仪表盘顶部提供筛选器，可以按以下条件筛选数据：
- 时间范围：近7天、近30天、自定义
- 平台：抖音、视频号、其他
- 商品类型：保健品、医疗器械、处方药

操作步骤：
1. 在筛选器区域选择相应条件
2. 点击"刷新数据"按钮或直接更改筛选条件
3. 系统将自动更新显示的数据

#### 关键指标查看
仪表盘展示以下关键指标模块：
- 千川投放：ROI基准线管控
- 智能剪辑：WASM多版本处理
- 达人推荐：带货条件匹配
- 发布调度：微信通知

查看详细信息：
1. 指标卡片加载完成后，会显示详细数据
2. 点击卡片可以查看更多信息

#### 数据可视化
系统提供多种图表类型：
- 折线图：展示趋势变化
- 柱状图：对比不同数据
- 饼图：显示占比关系

操作方法：
1. 图表区域自动加载数据
2. 鼠标悬停在图表上可查看具体数值
3. 部分图表支持点击交互

#### 智能选品
系统根据以下策略推荐选品：
- 舆情监控：过滤高风险商品
- 热点借势：选择上升期品类
- 数据验证：验证转化率等指标
- 利润测算：确保利润空间

查看选品策略：
1. 在"智能选品"区域查看推荐策略
2. 点击各策略卡片查看详情
3. 在"推荐选品"部分查看具体商品

#### 系统预警
预警系统监控以下状态：
- 正常：系统运行正常
- 预警：指标略有异常
- 异常：需要立即处理

处理预警：
1. 查看预警指示灯颜色（绿/黄/红）
2. 点击"查看详情"按钮
3. 根据提示采取相应措施

### 合作管理

#### 合作条款配置
1. 访问"/cooperation"页面
2. 点击"合作条款配置"标签
3. 填写合作基本信息：
   - 品牌方ID
   - 代销方ID
   - 商品ID列表
   - 合作类型
4. 配置详细条款：
   - 佣金比例
   - 库存管理
   - 售后服务
5. 点击"保存条款"按钮

#### 权益与责任管理
1. 在合作管理页面点击"权益与责任"标签
2. 查看当前条款版本
3. 修改条款：
   - 点击"修改条款"按钮
   - 填写修改内容
   - 提交修改请求
4. 处理争议：
   - 点击"提出争议"按钮
   - 填写争议详情
   - 提交仲裁申请

#### 合作流程跟踪
1. 点击"合作流程"标签
2. 查看当前合作状态
3. 操作流程：
   - 启动合作：点击"启动"按钮
   - 暂停合作：点击"暂停"按钮
   - 终止合作：点击"终止"按钮
   - 结算合作：点击"结算"按钮

#### 数据共享
1. 点击"数据共享"标签
2. 选择数据类型：
   - 销售数据
   - 用户反馈
   - 库存信息
3. 选择时间范围
4. 点击"获取数据"按钮
5. 查看和导出数据

### 流量运营

#### 千川广告投放
1. 访问"/traffic-operation"页面
2. 在"千川投放"区域填写：
   - 商品ID
   - 预算金额
   - 目标受众
   - 投放平台
3. 点击"生成投放计划"按钮
4. 查看ROI预测和预算分配
5. 确认并执行投放

#### 智能视频剪辑
1. 在流量运营页面找到"智能剪辑"区域
2. 填写视频信息：
   - 视频URL
   - 商品类型
   - 目标时长
3. 点击"开始处理"按钮
4. 等待处理完成
5. 查看高完播率视频版本
6. 选择合适的版本使用

#### 达人推荐
1. 在"达人推荐"区域填写：
   - 商品类型
   - 目标平台
   - 预算范围
2. 点击"获取推荐"按钮
3. 查看推荐达人列表
4. 点击达人姓名查看详细信息
5. 联系合适的达人进行合作

#### 发布调度
1. 在"发布调度"区域填写：
   - 发布平台
   - 发布时间
   - 主账号ID
   - 子账号ID列表
2. 点击"设置调度"按钮
3. 查看调度状态
4. 确认微信通知发送情况

### 风控与售后

#### 内容审核
1. 访问"/risk-control"页面
2. 在"内容审核"区域填写：
   - 内容类型
   - 内容文本
   - 商品ID
3. 点击"提交审核"按钮
4. 查看审核结果和风险等级
5. 根据建议修改内容

#### 药师转接
1. 在"药师转接"区域填写：
   - 用户问题
   - 商品信息
   - 用户信息
2. 点击"转接药师"按钮
3. 等待药师回复
4. 查看药师建议

#### 微信通知
1. 在"微信通知"区域填写：
   - 报备信息
   - Webhook URL
2. 点击"发送通知"按钮
3. 查看通知发送状态

#### 退换货审核
1. 在"退换货审核"区域填写：
   - 退货申请
   - 商品信息
   - 订单信息
2. 点击"自动审核"按钮
3. 查看审核结果
4. 根据结果处理退货

### 数据化运营

#### 指标查询
1. 访问"/data-operation"页面
2. 在"指标查询"区域选择：
   - 时间范围
   - 平台
   - 商品类型
   - 指标类型
3. 点击"查询指标"按钮
4. 查看详细指标数据

#### 仪表盘配置
1. 在"仪表盘配置"区域填写：
   - 图表列表
   - 时间范围
   - 平台
2. 点击"生成配置"按钮
3. 查看生成的仪表盘配置
4. 根据需要调整配置

#### 实时推送
1. 在"实时推送"区域填写：
   - Webhook URL
   - 指标阈值
2. 点击"设置推送"按钮
3. 系统将在指标异常时自动推送

## 常见问题

### 登录问题
**问题：无法登录系统**
解决方案：
1. 检查用户名和密码是否正确
2. 确认网络连接正常
3. 清除浏览器缓存后重试
4. 联系系统管理员重置密码

### 数据加载问题
**问题：仪表盘数据加载缓慢或失败**
解决方案：
1. 检查网络连接
2. 刷新页面重试
3. 更换时间范围或筛选条件
4. 联系技术支持

### 图表显示问题
**问题：图表无法正常显示**
解决方案：
1. 确认浏览器支持JavaScript
2. 刷新页面
3. 尝试使用其他浏览器
4. 检查是否有浏览器插件阻止脚本运行

### 功能操作问题
**问题：某些功能按钮无响应**
解决方案：
1. 确认已正确填写必填项
2. 检查网络连接
3. 刷新页面后重试
4. 联系技术支持
